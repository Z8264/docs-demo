        # 战术小组

        Forked from 《COUP》

        玩家：2～6 人

        时间：10～20 min

        规则：适中

        # 配件准备

        • 角色卡（15 张）：商人，刺客，盗贼，警长，守卫各 3 张。
        • 金币（若干）：准备一些标记卡，或指示物，代表金币。

        # 基本概念

        # 行动

        每个玩家轮流行动。行动阶段，玩家可以选择下表中任何一个行动，即使没有该角色。

        # 质疑

        任何角色的行动技能，包括阻止行动，可被质疑。基本行动（收入，交易，消灭）不可被质疑。被质疑的玩家，需要亮出拥有该行动技能的角色，如果没有，则质疑成功，玩家死亡一名角色。如果有，则质疑失败，质疑的玩家死亡一名角色。

        实战

        例如，玩家 A 选择执行投资（获得3金币），玩家 B 质疑 A，此时 A 需要亮出自己的商人角色牌。 如果没有商人，则质疑成功，A 死亡一个角色。 如果有商人，则质疑失败，玩家 B 死亡一个角色，玩家 A 将亮出的商人角色放回牌堆并重新抽取角色，同时继续执行创收，获得 3 金币。

        # 阻止

        阻止玩家的行动（可阻止的行动包括：交易，刺杀，偷窃）。阻止可被行动的玩家质疑。

        实战

        例如，玩家 A 选择偷窃玩家 B，B 阻止偷窃。 如果 A 不质疑 B 的阻止，则阻止成功，A 无法从 B 手中获取金币 。 如果 A 质疑 B 的阻止，则 B 需要亮出盗贼或警长。若质疑阻止成功：B 死亡一个角色，同时 A 从 B 手中获取 2 金币。若质疑阻止失败：B 更换亮出的角色，A 死亡一个角色，且 A 无法从 B 手中获取金币。

        # 角色死亡

        玩家选择手中的一个角色，正面朝上放在桌面上，直到游戏结束。死亡的角色不洗入牌堆。

        # 游戏规则

        # 准备阶段

        • 每个玩家抽取 2 张角色卡，背面朝上，仅自己可知
        • 每个玩家拿取 2 个金币

        # 行动阶段

        • 玩家行动，每回合只能执行一个行动。其他玩家只能选择质疑或阻止，不能同时选择。
        • 如果被质疑，结算质疑。(若质疑成功行动失败)。
        • 如果被阻止，行动玩家选择是否质疑其他玩家的阻止。结算阻止。(若不质疑阻止，或质疑阻止失败，则任务失败)。
        • 若行动成功，结算玩家的行动。

        注：

        如果玩家行动时，手中超过10个金币，必须执行“消灭”行动

        # 获胜条件

        • 最后未出局的玩家获胜（当玩家手里的角色全部死亡，该玩家出局）。

        # 常见问题

        刺杀行动，被质疑，失败如何结算？

        玩家 A 刺杀 玩家 B， B 质疑 A，A 亮出刺客，B 因质疑失败死亡一个角色。而 A 刺杀成功，B 还需要再死亡一个角色。因此，”刺杀被质疑失败“，被刺杀玩家会死亡 2 个角色。

        阻止刺杀被质疑成功如何结算？

        玩家 A 刺杀 玩家 B，B 阻止 A，A 质疑 B 的阻止，B 并没有守卫，此时 A 质疑成功，B 死亡一个角色。而 A 刺杀成功，B 还需要再死亡一个角色。因此，”阻止刺杀被质疑成功“，被刺杀玩家会死亡 2 个角色。

        刺杀失败还需要支付 3 金币吗？

        需要。

        # 模式扩展

        以下玩法模式仅供参考，你可以创造自己喜好的规则和朋友一起玩耍，玩的开心就好。

        # 吃鸡模式

        最后出局的玩家获胜。（参考以上基础规则）

        # 单挑模式

        2人游戏时，每名玩家准备阶段拿取一元金币。（上局游戏失败的玩家先行动）

        # 2V2模式

        A组2人，B组2人，按照ABAB的方式行动。玩家可以帮助队友质疑。（上局游戏的失败的队伍先行动）

        # 随从模式

        准备阶段玩家获得一张“村民”卡，当触发角色死亡时，玩家损失一张“村民”，而不是暗置的角色。这样可以增加游戏时间和容错。

        # 国战模式

        准备 A1，A2 B1，B2，C1，C2，6张阵营卡，玩家提前抽取，拿到（A1，B1，C1）的玩家公开身份，拿到（A2，B2，C2）的玩家知道出局才公开身份。 随机选择一个玩家开始行动，最后一个出局的队伍获胜。
